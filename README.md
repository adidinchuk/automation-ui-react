## Getting Started

### Installation

_Follow the steps below to build and run the application._


1. Clone the repo
   ```sh
   git clone https://github.com/adidinchuk/automation-ui-react
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build the app
   ```sh
   npm run build
   ```
4. Run the applicaton
   ```sh
   npm start
   ```
